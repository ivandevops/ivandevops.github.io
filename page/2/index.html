使用github+hexo搭建个人博客

发表于 2020-01-08 | 分类于 blog开发
这个世界上最遥远的距离，就是知道和做到

很久以前就有搭建博客的想法
奈何行动力不足再加上能力不够就没有去做
希望可以通过博客记录来自己的学习和生活
以下就是我搭建博客的一些心得
希望可以帮助到更多的人

准备工作

环境配置

  • windows操作系统
  • 下载安装git （用于将在本地写好的博客推送到github page，需要配置本地主机与远程git的连接）
  • 下载安装node.js （npm会默认被安装）
  • 下载安装hexo，cmd管理员模式运行 npm install -g hexo （可能需要翻墙）

本地初始化博客内容

  • 选定一个目录，新建一个文件夹作为博客的根目录
  • 进入文件夹内，打开git bash，输入hexo init
  • 运行hexo s，此时访问本地4000端口，博客启动，安装成功

关联github

  • 在github上创建名字为XXX.github.io的项目，xxx是自己的github用户名
  • 打开本地的博客根目录的_config.yaml，设置配置文件如下
  • 运行npm install hexo-deployer-git –save
  • hexo g 生成本地静态文件
  • hexo d 将本地静态文件推送到git
  • 此时打开浏览器访问，http://ivandevops.github.io 能看到博客页面代表关联成功

配置博客的博文与主题（持续更新）

首先介绍一些重要的命令

  • hexo g （当你配置好博客后，用于生成要推送到github上的内容）
  • hexo d （把生成的博客推送到github）
  • hexo s （本地启动，用于调试）
  • hexo new “myfirstblog” （会在source/_posts文件夹内生成myfirstblog.md,这个文件夹就是放博客的文件夹）

已完成的博文配置

  • title 文章主题

  • date 创建日期

  • tags 标签

  • categories 分类

  • 文章中添加居中模块

已完成的主题配置

  • 博客标题、博客描述（签名）、作者、语言、时区

  • 博客主题的自定义以及配置
